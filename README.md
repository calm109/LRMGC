# LRMGC

---

## Overview

**LRMGC** is a computational framework for single-cell RNA sequencing (scRNA-seq) data clustering. By integrating **low-rank matrix factorization** with **local graph regularization**, LRMGC effectively captures the intrinsic low-dimensional structures and local cell-to-cell transitions in heterogeneous cell populations.

---

## Getting Started

### Prerequisites

- MATLAB

### Installation

Clone this repository to your local machine:

```bash
git clone https://github.com/your_username/LRMGC.git
cd LRMGC
```
